# Get-Windows-Product-Key
Obtenha a chave do produto Windows. Este aplicativo é escrito em C# e destina-se a todos que precisam obter o número da chave do produto Windows ou 
para aqueles que desejam saber decodificar DigitalProductId exportado do registro.

Endereço de Registro: `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DigitalProductId`
